type: principle

Replace research/engineering jargon with the reader's mental model.

| Before | After |
|---|---|
| Evaluative | Researched |
| Cue-dominant | Used a shortcut |
| Informed | Evidence-grounded vote |
| Identifier-only | Identifier-only vote |
| Rolled off | Skipped the race |
| `R_top` / `R_mid` / `R_down` | Federal / Mid-tier / Down-ballot |

If you must use a technical term in a label, define it in a glossary
under the chart.
