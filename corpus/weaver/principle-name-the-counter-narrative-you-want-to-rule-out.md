title: Name the counter-narrative you want to rule out
type: principle
---

Any claim worth making has an obvious alternative explanation. Name it
early, then design the chart to either rule it out or acknowledge it
honestly.

Receipts: the Maplewood age pyramid's "28% male excess in 0–17" had
an obvious counter-narrative — *sampling noise at small geographies*.
Naming that counter-narrative forced the MOE whiskers, which resolved
the question visually. Without the naming step, the chart would have
implied a finding the data couldn't support.

If you can't name a plausible counter-narrative, either the claim is
too generic to be interesting, or you haven't thought about it hard
enough.
